# 我们是谁？

Linux User Group 简称LUG 可以译为 Linux用户组 是由洛理在校的 GUN/Linux 爱好者组建的社团。

# 我们要做的事

  • 组建专属洛理的Linux开源社区，提供信息交流共享的平台。
  • 宣传 国产 操作系统的使用，推广自由软件的应用。
  • 向校内外提供开源软件镜像、在线文档等服务。
  • 更多······

# 你可以学到什么？

  • Linux系统的各种骚操作，体验命令行的艺术，丢掉你的Windows吧
  • 想拥有自己的网站吗？来吧！我们从零基础教你。
  • 对硬件感兴趣，巧了，我们这里有各种派、开发板、路由器······
  • 玩游戏不过瘾，想自己做游戏，问题不大，我们会，点这里 (opens new window)体验一把小游戏。
  • 学校网络设施不友好，我们来开发！
